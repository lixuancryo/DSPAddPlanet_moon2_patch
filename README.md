# DSPAddPlanet-moon^2-patch

这是 Zincon 维护的《戴森球计划》DSPAddPlanet 个人 fork。

本项目基于 LittleSaya 的 DSPAddPlanet，并参考 Touhma 的 GalacticScale，由 Codex-GPT 协助制作。Zincon 本人是纯代码小白，项目能完成主要依靠原作者们的优秀工作和 AI 协助。

详细说明见：

- `DSPAddPlanet/README.md`
- 原版 DSPAddPlanet： https://github.com/LittleSaya/IndexOutOfRangeDSPMod/tree/master/DSPAddPlanet
- GalacticScale： https://github.com/Touhma/DSP_Galactic_Scale

核心改动：

- 支持通过 `OrbitAroundIndex` 添加卫星的卫星。
- 加入部分 GalacticScale 风格星球主题，可在不开启 GalacticScale 时由 DSPAddPlanet 使用。
- 增加是否允许这些主题进入随机星区生成的配置项。
- 删除或避开原版生成器中不稳定的小行星主题。
- 对 Beach、SulfurSea、GiganticForest 等主题做了兼容和视觉调整。

使用前请备份存档。
